# Security Policy

Bitte keine echten Zugangsdaten in Issues, Screenshots oder Beispieldateien veröffentlichen.

QuickGraph speichert keine API-Schlüssel. Falls eine importierte Datei dennoch ein nicht erkanntes Geheimnis enthält, führe sofort einen Virgin Reset durch, widerrufe den betroffenen Schlüssel beim Provider und melde das Erkennungsmuster ohne den Schlüssel selbst.

Öffentliche Sicherheitsmeldungen sollten nur reproduzierbare technische Details und anonymisierte Beispieldaten enthalten.
